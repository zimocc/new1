# 🎨 UI/UX 设计规范 (DESIGN STYLE)

## 一、核心风格：新丑风 (Neo-Brutalism)

应用采用了现代流行的 **Neo-Brutalism** 视觉风格，旨在通过鲜艳的色彩和大胆的几何线条吸引儿童注意力。

### 1. 核心设计元素
- **粗黑边框**: 所有卡片、按钮和容器均使用 `4px solid #3d405b` 的深色边框。
- **坚实投影**: 不使用模糊阴影，而是使用 0 或 4-6px 的实体位移投影（如 `box-shadow: 0 6px 0 #3d405b`）。
- **旋转动效**: 各种徽章（Badge）带有轻微的旋转角度（如 `-5deg` 或 `5deg`），营造活泼、非对称的趣味感。
- **纸面质感背景**: 全局背景使用圆点背景纹理 (`radial-gradient`)，模拟学生练习册的质感。

---

## 二、调色盘 (Color Palette)

应用使用了一套高饱和度、柔和的糖果色系：

| 颜色名称 | 十六进制 | 用途 |
|---------|----------|------|
| **深海军蓝** | `#3d405b` | 描边、主要文字、坚实投影 |
| **活力黄** | `#FEE685` | 教材卡片背景、选中态背景 |
| **蜜桃橙** | `#FFD6A7` | 统计卡片背景 |
| **薄荷绿** | `#96F7E4` | 进度卡片背景 |
| **天空蓝** | `#74b9ff` | 主按钮、自动发音徽章 |
| **珊瑚红** | `#ff7675` | 停止/危险操作按钮 |
| **浅灰蓝** | `#f0faff` | 全局页面背景 |

---

## 三、排版 (Typography)

- **主字体**: `'Microsoft YaHei', sans-serif`。
- **单词显示**: 
  - 动态字体大小调节：根据单词长度自动从 `55px` 降至 `24px`，确保长词不换行。
  - 装饰：使用 `3px 3px 0px #fdcb6e` 的黄色文字投影，增强视觉冲击力。
- **音标**: 保持灰色加粗 (`#636e72`)，确保清晰可读。

---

## 四、交互逻辑与微动效

### 4.1 按钮点击感
- **Active 状态**: 点击时按钮向下位移 (`translateY(5px)`)，同时投影消失，模拟真实按钮下压效果。
- **过渡**: 使用 `transition: all 0.1s ease` 确保响应极速。

### 4.2 词义反转
- **捉迷藏动效**: 中文释义默认隐藏在虚线框中。点击时使用 `cubic-bezier` 弹性过渡展开，增加互动的趣味性。

### 4.3 列表滚动
- **精准定位**: 切换单词或打开课前列表时，应用会自动计算 offset 并平滑滚动，确保当前学习项始终位于视野中心。
- **滚动条**: 自定义细窄灰色滚动条，避免遮挡内容。

---

## 五、响应式适配

- **核心宽度**: 针对手机屏幕优化，最大宽度限制为 `600px`。
- **触摸热区**: 
  - 在单词卡片左右两侧设计了 **25% 宽度的隐形点击区**，方便单手大拇指左右滑动切换。
  - 所有按钮高度均大于 `44px`，符合人体工程学。
