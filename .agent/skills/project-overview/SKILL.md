## 一、项目概述

- **项目名称**: 背单词 (new1)
- **定位**: 面向儿童（小学 & 新概念英语）的移动端优先 H5 单词学习工具
- **线上部署**: Cloudflare Pages（含 Functions Serverless API 代理）
- **核心价值**: 提供"听写模式"、"自动发音"、多教材切换和持久化学习进度等功能
